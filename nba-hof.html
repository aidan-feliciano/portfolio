NBA Hall of Fame Prediction — 2025

Problem

Can we predict whether an NBA player will be inducted into the Hall of Fame from career statistics? Because the dataset is highly imbalanced (~3% HoF), overall accuracy is misleading. I evaluate models with PR-AUC and F1 for the HoF class, and use threshold tuning to balance precision and recall.

Data

Player-level career stats (totals + advanced metrics): pts, ast, trb, g, ws, per, efg%, etc. I engineered per-game features (e.g., pts_per_g) and used a signed log for career_ws to handle negatives. Train/test split is stratified to preserve the HoF ratio.

Methods

  • Baselines: Logistic Regression (scaled), class-weighted LR, SMOTE + LR
  • Feature Engineering: per-game rates, log transforms (signed log for Win Shares)
  • Tree Models: Gradient Boosting, Random Forest
  • Threshold Tuning: choose operating point to maximize F1 on HoF class
  • Metrics: PR-AUC, ROC-AUC, Precision/Recall/F1 (HoF), overall Accuracy
Why PR-AUC? With heavy class imbalance, PR-AUC better reflects minority-class performance than accuracy or ROC-AUC alone.

Results (Top Models)

Model Variant / Threshold Accuracy PR-AUC Precision@HoF Recall@HoF F1@HoF
Gradient Boosting FE, tuned threshold ≈ 0.53 0.983 0.720 0.724 0.724 0.724
Logistic Regression SMOTE + FE, tuned threshold ≈ 0.97 0.983 0.659 0.724 0.724 0.724
Logistic Regression Baseline (scaled), threshold = 0.50 0.983 0.695 0.760 0.655 0.704

The best balanced performance came from Gradient Boosting + Feature Engineering at a tuned threshold of ~0.53, achieving Precision ≈ Recall ≈ F1 = 0.724 on the HoF class while maintaining 98.3% overall accuracy. A tuned SMOTE + LR approach matched this F1, offering an interpretable linear baseline. Untuned class-weighted LR favors recall at the expense of precision, illustrating the value of threshold selection.

Takeaways

  • Accuracy alone overstates performance on imbalanced problems; PR-AUC and F1@HoF are more informative.
  • Feature engineering (per-game stats, careful transforms) improves separability across models.
  • Threshold tuning is crucial—small shifts trade precision for recall and can optimize F1 for the target class.
  • Tree models (GB) provide strong PR-AUC with modest tuning; LR remains a solid interpretable baseline.

Tools

  • Models: Logistic Regression, Gradient Boosting, Random Forest
