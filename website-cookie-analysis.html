Website Cookie Analysis - December 2024

Problem

How can businesses utilize website cookie data to better understand, target, and extract revenue from their online customer base? This project aims to understand and analyze behavior patterns, revenue drivers, and predictors of customer transactions.

Dataset Overview

We used anonymized cookie-level web data from an online store. The data spans from 8/1/2010 to 1/31/2011. There are over 16,000 records, with each representing a "website session," where the online store tracks user activity on the website. To clarify, a cookie is a packet of data sent by a web server to a browser and then returned back by the browser each time the user subsequently accesses the same server. Data Preparation & Cleaning

  • Dropped columns where all values were either 1 or 0 (e.g., Nuber of Records, goalCompletionsAll).
  • Removed ~200 rows with missing values.

Exploration & Segmentation

  • There were 3,570 different cities representing 697 different regions
  • Top cities included New York, Moscow, Kyiv, and London.
  • Users were segmented by engagement level (based on time on site):
    • Low: 0–28 min
    • Medium: 29–87 min
    • High: 87+ min

Search Engines & Revenue

We ran a multiple linear regression on organic traffic to determine search engines' impact on revenue. Key findings:

  • Google had a significant positive effect, adding ~$9.37 per visit.
  • Each visit added ~$15.86 to revenue; each bounce lost ~$19.55.

Predicting Transactions

We tested both logistic regression and random forest to predict whether users would make a transaction.

  • Logistic regression was chosen for simplicity, with an accuracy of 89.81%.
  • Significant predictors: medium (organic), new visits, visits (+); bounces (–).
  • Organic users were 1.5× more likely to transact than others.

Scoring & Targeting

Using predicted probabilities, we prioritized users for targeting:

  • Top 102 users delivered ~100 transactions (limited supply rule).
  • Top 953 users had a transaction rate above 50% (marginal supply rule).

Business Insights

  • Focus SEO efforts on Google keywords and rankings.
  • Invest in partnerships and re-engagement campaigns to drive visits.
  • Improve site experience to reduce bounce rates (chatbots, mobile-first design).
