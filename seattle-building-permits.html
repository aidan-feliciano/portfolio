Seattle Building Permits Analysis

Problem

How is construction activity distributed across Seattle? We aimed to understand where, what, and how building projects are happening across the city using permit data to uncover trends and draw conclusions for real estate stakeholders.

Data Source

We used 2023 building permit data from the City of Seattle Open Data Portal. The dataset contains 5,596 permits with attributes like project cost, housing units, contractor info, zip code, and more.

Goal / Key Metrics

  • Identify high-growth zip codes
  • Compare commercial vs. residential permit activity
  • Estimate project costs, timelines, and contractor trends

Actions

  • Cleaned messy fields using SQL (e.g., correcting shifted columns)
  • Explored distributions in Python with pandas & matplotlib
  • Grouped by zip, permit class, contractor, and cost brackets

Results

  • Zip codes 98101, 98104, and 98109 had the most commercial activity
  • 50% of permits approved within 50 days; average construction time = 163 days
  • Most contractors only handled one project; a few did 3+

Insights and Next Steps

  • Homebuyers: focus on low-permit areas for quieter neighborhoods
  • Developers: prioritize zip codes with high average project cost and fast approvals
  • City planners: use permit density to target infrastructure growth
