The Enchantments Lottery - March 2025

Background

The Enchantments is a region in Washington state's Cascade Mountain Range that is home to over 700 alpine lakes and ponds surrounded by a vast expanse of mountain peaks. It is one of the most popular outdoor destinations for hikers, campers, rockclimbers, and anyone who enjoys the wilderness. As a result of the popularity, the Washington Trails Association introduced a lottery system to reduce the overcrowding that the region was seeing.

How the Lottery Works

The lottery opens on February 15th and closes On March 1st each year. Each application consists of three entries in order of preference. An individual entry comprises a zone, date, and group size. The preference is determined by order in the application. For example, one application might look like:

  1. Zone: Colchuck Zone, Entry Date: 5/15/2024, Group Size: 4
  2. Zone: Core Enchantment Zone, Entry Date: 8/7/2024, Group Size: 7
  3. Zone: Eightmile Zone, Entry Date: 7/1/2024, Group Size: 3
An applicant can only apply once, which greatly reduces the odds of an individual winning the lottery. However, winning the lottery is not completely up to chance; there are some factors that can dramatically change your odds of wining the lottery. This project provides some critical insights of how to increase your odds to win the lottery, and it provides some recommendations for how to completely maximize your chances.

Data Overview

There were two sets of data used in this project: the 2024 Enchantment lottery statistics, and the historical weather data for the area. Each observation in the lottery statistics represents an application (like the example above), and it is structured as follows:

  • Lottery Year: the year in which the application was submitted (2024 for all of our data)
  • LotteryID: the ID associated with the application
  • Choice ID: the ID associated with each entry for the application (there will be 3 per 1 LotteryID)
  • Choice Number: the order ranking of each entry in the application (1, 2, and 3 per application)
  • Preferred Entry Date: the preferred entry date specified for each application entry
  • Preferred Zone: the preferred zone specified for each application entry
  • Maximum Requested Group Size: the requested group each application entry
  • Choice Awarded?: whether or not the individual entry of the applicaiton was awarded
  • Application Result: whether or not any one of the three entries in the application
So, for the same example above, the data would look like this:
Lottery Year LotteryID Choice ID Choice Number Preferred Entry Date Preferred Zone Maximum Requested Group Size Choice Awarded? Application Result
2024 2024_44 2024_44_1 1 5/15/2025 Colchuck Zone 4 Yes Awarded
2024 2024_44 2024_44_2 2 8/7/2025 Core Enchantment Zone 7 No Awarded
2024 2024_44 2024_44_3 3 7/1/2025 Eightmile Zone 3 No Awarded
In this example, the first application entry was awarded, which makes the overal application successful. So, even though the second and third entries were not awarded, they are still part of a successful application.
